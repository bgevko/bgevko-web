### GET
```bash
curl https://example.com
```

### POST
```bash
# Simple request with no body
curl -X POST http://localhost:3000/confirm
```
```bash
# Headers and JSON body
curl -X POST http://localhost:3000/confirm -H "Content-Type: application/json" -d '{"key":"value"}'
```
```bash
# Send data from a file
curl -X POST http://localhost:3000/confirm -d @path/to/json
```
